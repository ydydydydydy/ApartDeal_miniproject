## 개요
ApartDeal_Project는 500만 건의 전국 아파트 매매 거래 데이터를 통해 수도권 아파트 데이터를 분석한 프로젝트 입니다

###### 기간: 2024.1.01 ~ 2024.1.03 (3일)

###### 주요 기능 및 특징:

1. 분석 목표
2. 분석 방법
3. 사용한 라이브러리
4. 활용한 데이터 셋
5. 분석 결과 시각화
6. 분석 결과 정리
7. 데이터 출처

#### 역할

- 데이터 분석 & 시각화
- html/css
